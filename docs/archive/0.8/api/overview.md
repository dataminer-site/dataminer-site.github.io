There are various client APIs for dataminer. dataminer's "native" API is [C++](cpp), with "official" wrappers available for [C](c/overview), [Python](python/overview), [R](r), [Java](java), [Node.js](nodejs/overview), [WebAssembly/Wasm](wasm/overview), [ODBC API](odbc/overview), [Julia](julia), and a [Command Line Interface (CLI)](cli).

There are also contributed third-party DataMiner wrappers for [Ruby](https://github.com/suketa/ruby-dataminer), [Go](https://github.com/marcboeker/go-dataminer), [C#](https://github.com/Giorgi/dataminer.NET), [Rust](https://github.com/wangfenjin/dataminer-rs) and [Common Lisp](https://github.com/ak-coram/cl-dataminer).
