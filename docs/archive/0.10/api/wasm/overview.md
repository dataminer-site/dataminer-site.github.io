DataMiner has been compiled to WebAssembly, so it can run inside any browser on any device.

<!-- markdownlint-disable-next-line -->
{% include iframe.html src="https://shell.dataminer.org" %}

dataminer-Wasm offers a layered API, it can be embedded as a [JavaScript + WebAssembly library](https://www.npmjs.com/package/@dataminer/dataminer-wasm), as a [Web shell](https://www.npmjs.com/package/@dataminer/dataminer-wasm-shell), or [built from source](https://github.com/dataminer-site-wasm) according to your needs.

## Getting Started with dataminer-Wasm

A great starting point is to read the [dataminer-Wasm launch blog post](/2021/10/29/dataminer-wasm)!

Another great resource is the [GitHub repository](https://github.com/dataminer-site-wasm).

For details, see the full [dataminer-Wasm API Documentation](https://shell.dataminer.org/docs/modules/index.html).
